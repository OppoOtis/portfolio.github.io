Midi Controller

What's the project

This is a school project where we had to create something with an arduino. I really like music so I wanted to create a custom midi controller. This controller was largely inspired by the theramin instrument.

The process

The first step was the design process where i had to check what component i should use and how they would work in the system. After that I made each component work on their own and wrote some code for it. Later on I esambled everything together into a mdf printed casing. The last step was to be able to send midi data to the computer or a sinth to play sounds/notes.
