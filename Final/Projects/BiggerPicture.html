Bigger Picture

What's the project

This project was made for a client that had a problem with scientists not understanding each other. The goal of this game is to make scientist talk more with ech other, and maybe even get to understand their standpoints. As the only designer and developer in the group my task was to playtest as much game ideas as possible with my group. And after the final concept, programm and make the game work.

The process

At the start we made multiple concept, but the client was not happy with the ideas we had. So in the end we combined the mechanics the client did like into one concept. This concept required a lot of programming and networking so that was my main task after concepting. I learned a lot about networking as I had never done that before. And also learned about presenting and communicating with a client.
