Run

What's the project

This was a school project called "Game Feel", where we had to change an already existing game. The goal was to make the game litteraly feel good to play, hence "Game Feel". The game itself was inspired by Canabalt and its cool running. This project was made in about 2 weeks.

The process

The first step I worked on was to implement subtle mechanics like different jump heights, cayote time and slowly speeding up the running. After that i mainly worked on visuals, particles, animation, music and sound design. In the end I added some simple enemies but those are not implemented that well.
